# 当前建议的发版顺序

先在仓库根目录确认校验通过：

```bash
npm run typecheck
npm run build
npm run pack:check
npm run smoke:install
```

然后按这个顺序发版：

```bash
npm publish --workspace @statekit-vue/shared --access public
npm publish --workspace @statekit-vue/vue --access public
```

发布后建议立刻检查：

```bash
npm view @statekit-vue/shared version
npm view @statekit-vue/vue version
```

# 发版前最后再看一遍的点
